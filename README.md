1. Click **Use this template** → **Create a new repository**.
2. The new repo will contain provider-native guidance for Cursor, Claude,

## Customisation

For full control over objectives, strictness, and language settings,
install the `agent-rules` CLI:

```bash
uvx --from git+https://github.com/climate-x-org/agent-rules agent-rules \
  render --dest . --language typescript
```

See the [agent-rules README](https://github.com/climate-x-org/agent-rules#readme)
for all options.
